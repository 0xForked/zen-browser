## ðŸš€ Run Locally

Clone the project

```bash
git clone https://github.com/zen-browser/desktop.git
cd desktop
```

Install dependencies 

```bash
npm i
```

Download and bootstrap the browser

```
npm run init
```

Start building the browser

```
npm run build
```

Finally, run the browser!

```
npm start
```
